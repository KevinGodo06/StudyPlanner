# StudyPlanner

StudyPlanner è un gestionale scolastico web per studenti.
Organizza i propri impegni all'interno di un calendario, e le valutazioni scolastiche all'interno di vari grafici suddivisi per materia.

É possibile accedere tramite google, oppure nella maniera più classica con e-mail e password.

studyplanner.altervista.org
